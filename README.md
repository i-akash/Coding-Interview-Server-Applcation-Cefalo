## Docs

#### Create Problem Tab

- Input Constraint ( `format : json` )

  - Example

    ```json
    {
      "arrayLength": "int",
      "array": "[string]"
    }
    ```

    ```json
    {
      "str1": "string",
      "str2": "string"
    }
    ```

    ```json
    {
      "a": "float",
      "b": "int"
    }
    ```

## Features
- code selection, inline code format, emoji
- custom input and test case support with python and js
- multiple room support
- public and private room
